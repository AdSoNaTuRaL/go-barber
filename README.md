# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/AdSoNaTuRaL/go-barber.git
```
### üì¶ Run API

```bash
# Go to server folder
$ cd go-barber/back-end

# Install Dependencies
$ yarn

# Run Aplication
$ yarn dev:server
```
Access API at http://localhost:3333/

### üíª Run Web Project

```bash
# Go to web folder
$ cd go-barber/front-end

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.

### üì± Run Mobile Project

```bash
# Go to mobile folder
$ cd go-barber/mobile

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start

$ yarn android
```
